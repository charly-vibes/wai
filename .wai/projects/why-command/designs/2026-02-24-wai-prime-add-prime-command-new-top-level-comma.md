wai prime (add-prime-command): new top-level command for session orientation. Auto-detects project (same logic as wai close). Reads most recent handoff from .wai/projects/<project>/handoffs/*.md, extracts YAML frontmatter date + first paragraph as 80-char snippet. Calls plugin status hook (same as wai status) for per-plugin one-line summaries. Invokes bd ready for highest-priority ready issue as suggested next. Omits handoff line when no handoff, omits suggested-next when no ready issues. Output format: header bullet lines for project/handoff/plugins, arrow for suggested next. Openspec: openspec/changes/add-prime-command/.
